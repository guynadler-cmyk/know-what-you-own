You are building a feature for an existing stock research web app called "Know What You Own".

The app already stores structured company analysis data from SEC filings including:
- moat
- themes
- strengths
- valuation_score
- quality_score
- quadrant
- marketCapBucket
- embeddings

These are stored in a database table called:

companyDiscoveryIndex

Goal
Build a STOCK DISCOVERY / SCREENER feature where users can discover companies by selecting tags like:

Enterprise Software
Recurring Revenue
AI
Cybersecurity
Marketplace
Network Effects
High Switching Costs
Developer Tools
Cloud Infrastructure
Data Platforms

Users should be able to:

1. Select tags
2. Drag-and-drop tags into a filter box
3. Click SEARCH
4. Get companies with similar attributes from the database.

Think through the feature step-by-step before coding.

--------------------------------

Step 1 — UX Design

Design a modern UI similar to a SaaS analytics dashboard.

Page layout:

Header
"Discover Companies"

Left Panel
Tag Library (draggable)

Tags appear as pills:

[Enterprise Software]
[Recurring Revenue]
[AI]
[Cybersecurity]
[Developer Tools]
[Cloud]
[Marketplace]
[Network Effects]
[High Margins]

Users drag tags into a box called:

"Your Filters"

Inside the filter box selected tags appear as removable chips.

Example:

Your Filters
[Enterprise Software] [Recurring Revenue] [High Margin]

Search Button

--------------------------------

Step 2 — Search Logic

When user clicks SEARCH:

Query companyDiscoveryIndex where

themes overlap selected tags
OR
moat overlap selected tags

Sort results by:

1. qualityScore descending
2. valuationScore descending

Return:

ticker
companyName
themes
moat
qualityScore
valuationScore
quadrant

Tailwind

Node / Express

Database query example:

SELECT *
FROM companyDiscoveryIndex
WHERE themes && ARRAY[selectedTags]
OR moat && ARRAY[selectedTags]
