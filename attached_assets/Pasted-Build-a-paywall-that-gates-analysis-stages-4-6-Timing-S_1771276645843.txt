Build a paywall that gates analysis stages 4-6 (Timing, Strategy, Protection) behind an email capture system with a "Friday Report → Monday Actions" value proposition. The paywall is shown to ALL users who have not yet signed up. No A/B testing — every visitor sees the gate until they provide their email. Here is the complete specification:

OVERVIEW: Three-state paywall with two-tier gating

States: locked (default), skipped (user dismissed the initial modal), unlocked (email verified in database).

Stages 1-3: No paywall, always accessible.
Stage 4 (locked): Blurred content + floating "Friday Report" modal with skip option.
Stage 4 (skipped): Full content visible + compact dismissible email banner at top.
Stage 5+ (locked): Blurred content + floating "action gate" modal with TWO signup paths (Friday Report or Restnvest Waitlist). No skip option.
Stage 5+ (skipped): Hard gate — action gate modal only, no content behind. User must sign up or go back to earlier stages.
Any stage (unlocked): Full content, no paywall at all.
Unlock check: On page load, if rn_user_email exists in localStorage, call GET /api/waitlist/check?email={email} to verify the email exists in waitlist_signups. If verified, mark the ticker as unlocked. If not verified (or no stored email), show the paywall.

STEP 1: Create client/src/lib/abTest.ts — Paywall state management (no A/B logic).

const PAYWALL_UNLOCK_PREFIX = "rn_unlocked_";
const PAYWALL_SKIPPED_PREFIX = "rn_skipped_";
const USER_EMAIL_KEY = "rn_user_email";
export type PaywallState = "locked" | "skipped" | "unlocked";
export function getPaywallState(ticker: string): PaywallState {
  const key = ticker.toUpperCase();
  if (localStorage.getItem(`${PAYWALL_UNLOCK_PREFIX}${key}`) === "true") return "unlocked";
  if (localStorage.getItem(`${PAYWALL_SKIPPED_PREFIX}${key}`) === "true") return "skipped";
  return "locked";
}
export function isPaywallUnlocked(ticker?: string): boolean {
  if (!ticker) return false;
  return getPaywallState(ticker) === "unlocked";
}
export function unlockPaywall(ticker: string): void {
  const key = ticker.toUpperCase();
  localStorage.setItem(`${PAYWALL_UNLOCK_PREFIX}${key}`, "true");
  localStorage.removeItem(`${PAYWALL_SKIPPED_PREFIX}${key}`);
}
export function unlockAllTickers(): void {
  // Called after verifying email exists in database — unlock any ticker
  // The next getPaywallState call for any ticker with rn_unlocked_ set will return "unlocked"
}
export function skipPaywall(ticker: string): void {
  const key = ticker.toUpperCase();
  localStorage.setItem(`${PAYWALL_SKIPPED_PREFIX}${key}`, "true");
}
export function shouldShowPaywall(stage: number): boolean {
  return stage >= 4;
}
export function getStoredEmail(): string | null {
  return localStorage.getItem(USER_EMAIL_KEY);
}
export function setStoredEmail(email: string): void {
  localStorage.setItem(USER_EMAIL_KEY, email.toLowerCase().trim());
}

Add a new route that checks if the given email exists in the waitlist_signups table (case-insensitive). Returns { exists: true/false }. This is used on page load to auto-unlock returning users.

// In routes.ts, add:
app.get("/api/waitlist/check", async (req, res) => {
  const email = (req.query.email as string || "").toLowerCase().trim();
  if (!email) return res.json({ exists: false });
  
  try {
    const [match] = await db
      .select({ email: waitlistSignups.email })
      .from(waitlistSignups)
      .where(eq(waitlistSignups.email, email))
      .limit(1);
    res.json({ exists: !!match });
  } catch (error) {
    res.json({ exists: false });
  }
});

Auto-unlock on page load — Add a useEffect that runs when currentTicker changes:

useEffect(() => {
  if (!currentTicker) return;
  // First check localStorage
  const localState = getPaywallState(currentTicker);
  if (localState === "unlocked") {
    setPaywallState("unlocked");
    return;
  }
  // Then check if stored email exists in database
  const storedEmail = getStoredEmail();
  if (storedEmail) {
    fetch(`/api/waitlist/check?email=${encodeURIComponent(storedEmail)}`)
      .then(r => r.json())
      .then(data => {
        if (data.exists) {
          unlockPaywall(currentTicker);
          setPaywallState("unlocked");
        } else {
          setPaywallState(localState); // "locked" or "skipped"
        }
      })
      .catch(() => setPaywallState(localState));
  } else {
    setPaywallState(localState);
  }
}, [currentTicker]);
