Implement a desktop-first "Discovery Integration" on the Stock Research / Analysis page.

Context

Users currently analyze a company (example: NVIDIA) and see structured sections such as:

• Investment Thesis
• Business Overview
• Competition
• Market Opportunity
• Value Creation

These sections already contain structured tags such as:

Investment Themes
AI/ML
Cloud Infrastructure
Gaming

Competitive Moats
Network Effects
Brand Power
Patents

Market Opportunity
TAM $500B
Growing 20% YoY

Value Creation
High Margins 70%+
Platform Economics
Recurring Revenue

Goal

Allow users to seamlessly discover other companies using these tags directly from the research page.

We will optimize for desktop only for now.

Mobile layout can remain unchanged.

------------------------------------------------

Feature 1 — Discovery Trigger Button

Add a primary button under the Investment Thesis panel.

Example:

[ Discover Companies Similar to NVIDIA ]

When clicked:

1. Collect all visible thesis tags
2. Open the discovery page
3. Pre-apply these tags as filters

Example generated URL:

/discover?tags=AI/ML,Cloud Infrastructure,Network Effects,Platform Economics

Implementation logic:

const tags = Array.from(document.querySelectorAll(".tag-chip"))
  .map(tag => tag.dataset.tag);

const uniqueTags = [...new Set(tags)];

const url = `/discover?tags=${encodeURIComponent(uniqueTags.join(","))}`;

window.open(url, "_blank");

Feature 6 — Smart Similarity Discovery

When discovery is triggered from a company page:

1. Extract all tags
2. Deduplicate
3. Rank other companies based on shared tags

Ranking logic:

similarity_score =
(shared_tags / total_tags) * 100

Example result:

AMD — 82% Similar
Snowflake — 75% Similar
Microsoft — 71% Similar
